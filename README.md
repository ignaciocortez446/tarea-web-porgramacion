# tarea-web-porgramacion


¡Bienvenido al repositorio oficial del proyecto **Desafío Web 2005**! Esta suite de aplicaciones web demuestra cómo las matemáticas abstractas y los algoritmos computacionales se aplican directamente para resolver problemas críticos del mundo real, tales como la planificación financiera, la criptografía asimétrica y la ciberseguridad avanzada.

Diseñado con un enfoque moderno, responsivo (adaptable a PC y dispositivos móviles) y con identidades visuales independientes de alta fidelidad tecnológica.

Modelo de Planificación Financiera (Sucesión de Fibonacci)Ubicación: modelo-fibonacci/Estética: Fintech Esmeralda (Moderna interfaz bancaria).Problema que resuelve: Combate la monotonía y la falta de motivación del ahorro tradicional de cuotas fijas utilizando un modelo de aceleración financiera dinámico.Lógica Avanzada: Implementa la secuencia iterativa recursiva analítica elemental ($A_n = A_{n-1} + A_{n-2}$) sin consumir arrays en memoria, calculando proyecciones complejas en tiempo real y formateando los resultados a moneda local (Bs.).

🔒 2. Modelo de Validación Criptográfica (Números Primos)
Ubicación: modelo-primos/

Estética: Terminal Cyberpunk Hacker (Colores oscuros con contrastes naranja neón).

Problema que resuelve: Simula un módulo de firewall que evalúa la fortaleza de códigos de acceso contra ataques masivos por diccionario.

Lógica Avanzada: El núcleo ejecuta un análisis atómico de divisores enteros positivos para verificar el aislamiento criptográfico del número ingresado. Si el residuo exacto es igual a dos, se aprueba como token seguro, emitiendo alertas visuales dinámicas según el nivel de vulnerabilidad.

🧬 3. Modelo de Criptografía Híbrida Avanzada (Fibonacci + Primos)
Ubicación: modelo-hibrido/

Estética: Laboratorio Quantum (Estilo futurista de ciencia ficción en tonos violeta).

Problema que resuelve: Mitiga la predictibilidad de los tokens de un solo uso (OTP). Los patrones secuenciales simples son fáciles de hackear; este modelo combina dos leyes matemáticas para generar irregularidad estadística.

Lógica Avanzada: Fusiona ambos algoritmos en un bucle cerrado de alto rendimiento. Genera la serie Fibonacci y, de forma simultánea, aplica un filtro selectivo estricto para aislar únicamente los "Primos de Fibonacci", simulando un generador de claves de grado militar.

🚀 Características Técnicas Destacadas
Diseño UI/UX Futurista: Cada página cuenta con una paleta de colores inmersiva diseñada desde cero a través de variables CSS (:root), efectos de profundidad de tarjetas, sombras difuminadas y estados :hover animados.

100% Responsivo: Adaptabilidad total testeada mediante Media Queries. La navegación se reestructura limpiamente si se visualiza en monitores de PC de escritorio, laptops o pantallas táctiles de smartphones.

Optimización de Código: Los motores en JavaScript se ejecutan de manera nativa (Vanilla JS) optimizando el rendimiento del navegador al evitar el uso redundante de colecciones pesadas u objetos globales.
